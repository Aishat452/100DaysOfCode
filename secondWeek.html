HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used ro define our page content, structure and meaning.
You don't for styling purposes. Use CSS for that instead

  • HTML uses "elements" to describe (anotate) content
  • HTML elements typically has a opening tag, contentand a closing tag
  • You can also have a void (empty) elements like images (img)
  • You can also configure elements with attribute
  • There's a long list of aailable element, but you'll gain experience over time. No worries

Learn more about all available HTML elements on the MDN HTML elements reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplications, you typically use global styles (e.g via the "style" element) instead.
  • Alternatively, you can work with external style sheet files which you "link" to.
  • When working with CSS, concepts like "inheritance", "specificity" and "box-model" should be well understood

Learn more about all available CSS properties and values on the MDN CSS property reference.
